# Real-Time Chat App

This is a real-time chat application built using the MERN stack and Socket.IO. The project allows users to communicate instantly with secure authentication and a simple user interface.

## Features

- User Signup and Login
- JWT Authentication
- Real-Time Chat using Socket.IO
- Send and Receive Messages
- Image Sharing
- Online User Status
- Secure Cookies
- Responsive UI

## Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Axios

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- Socket.IO
- JWT
- Cloudinary

## Project Structure

```
frontend/
backend/
```

## Installation

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

## Environment Variables

Create a `.env` file inside the backend folder and add the required environment variables.
